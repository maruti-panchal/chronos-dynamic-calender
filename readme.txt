# Chronos Calendar

Chronos Calendar is a modern, feature-rich calendar application designed for professionals and teams to manage schedules, events, and reminders with ease. Built with React, Next.js, and Tailwind CSS, it offers a beautiful UI, multiple calendar views, and advanced event management.

## 🌐 Live Demo

Check out the live demo: https://chronos-dynamic-calender.vercel.app/


## ✨ Features

- **Multiple Views:** Switch between Month, Week, Day, and Agenda views.
- **Color Categories:** Organize events with customizable color-coded categories.
- **Recurring Events:** Set up events that repeat daily, weekly, monthly, or yearly.
- **Reminders:** Get notified before events with flexible reminder options.
- **Powerful Search:** Quickly find events by title, description, or location.
- **Responsive Design:** Optimized for desktop and mobile devices.
- **Confetti & Animations:** Delightful UI touches for a joyful experience.

## 🛠️ Tech Stack

- **React 19**
- **Next.js 15**
- **Tailwind CSS**
- **Radix UI**
- **Framer Motion**
- **Lucide Icons**
- **Date-fns**

## 📦 Installation

```sh
pnpm install
```

## 🚀 Development

Start the development server:

```sh
pnpm dev
```

## 🏗️ Build

Build for production:

```sh
pnpm build
```

## 📁 Project Structure

- `app/` – Next.js app entry, global styles
- `components/` – UI components and calendar views
- `lib/` – Utilities and sample data
- `public/` – Static assets
- `styles/` – Global CSS

## 🖥️ Usage

1. Clone the repository.
2. Install dependencies.
3. Run the development server.
4. Open [http://localhost:3000](http://localhost:3000) in your browser.


## 📚 Customization

- Edit `lib/sample-data.js` to change sample events and categories.
- Tweak styles in `app/globals.css` and `tailwind.config.js`.
